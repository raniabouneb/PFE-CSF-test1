# Police Artigo

Pour afficher **Conseil**, **Solution** et **Formation** en police Artigo, placez ici les fichiers .woff2 suivants :

- `Artigo-Regular.woff2`
- `Artigo-RegularItalic.woff2`
- `Artigo-SemiBold.woff2`
- `Artigo-SemiBoldItalic.woff2`

Vous pouvez les obtenir via [Nova Type Foundry](https://novatypefoundry.com/fonts/artigo/) ou [Adobe Fonts](https://fonts.adobe.com/fonts/artigo) (exporter en .woff2 si besoin). Les noms des fichiers peuvent varier selon la fonderie ; adaptez les chemins dans `app/globals.css` si nécessaire.
